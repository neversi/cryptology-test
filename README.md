# Cryptology test

Test problem for cryptology

## Description

Simple DoM storage with mock processing.
DoM storage is implemented with Binary Search Tree (array representation). While simple implementation with sorting exists.

## Dependencies

- go 1.20
- make (optional)

## How to setup

To build binary:

```shell
make build
```

To run tests

```shell
make test
```

To run benchmarks

```shell
make bench
```
